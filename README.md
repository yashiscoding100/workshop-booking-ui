# Workshop Booking UI/UX Enhancement

## Overview
This project improves the UI/UX of the workshop booking system with a cleaner and more modern design.

## Design Principles
- Simple and clean layout
- Better spacing and readability
- Consistent colors

## Responsiveness
- Used Bootstrap for mobile-friendly layout
- Improved spacing for small screens

## Trade-offs
- Focused on UI instead of backend changes
- Kept design simple for performance

## Challenges
- Understanding Django templates
- Navigation issues
- Solved by simplifying navbar

## Setup Instructions

git clone https://github.com/yashiscoding100/workshop-booking-ui.git
cd workshop_booking
pip install -r requirements.txt
python manage.py runserver

Open:
http://127.0.0.1:8000
